# RetailShield

RetailShield is a smart retail management system designed to help store owners monitor staff, manage operations, and improve overall efficiency. The project includes a frontend interface and a backend server for handling logic and data.

---

## Features

* Staff Management
* Admin Dashboard
* Store Operations Handling
* Secure Backend APIs
* Simple and responsive UI

---

## Tech Stack

### Frontend:

* HTML
* CSS
* JavaScript

### Backend:

* Node.js
* Express.js

## Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/RetailShield.git
cd RetailShield
```

### 2. Install backend dependencies

```
cd backend
npm install
```

### 3. Run the server

```
node server.js
```

### 4. Open frontend

Open `index.html` in your browser
(or use Live Server in VS Code)

---

## Environment Variables

Create a `.env` file in the backend folder and add:

```
PORT=5000
DATABASE_URL=your_database_url
```

---

## Notes

* `node_modules` is not included in the repository
* `.env` file is not uploaded for security reasons
* Backend will not run on GitHub Pages (frontend only)
